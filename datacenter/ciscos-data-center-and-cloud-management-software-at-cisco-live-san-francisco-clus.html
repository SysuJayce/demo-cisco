Cisco’s Data Center and Cloud Management Software at Cisco Live San Francisco #CLUS

Carolina Fernandez - May 18, 2014 - 0 Comments

We’re excited to showcase new innovations in our data center infrastructure automation and cloud management software at Cisco Live San Francisco this week!

This past Friday, Cisco announced the new Cisco UCS Director version 5.0 – including Application Centric Infrastructure (ACI) support and integration with the Cisco Application Policy Infrastructure Controller (APIC). Cisco customers, prospective customers, and partners in the data center market won’t want to miss these two Cisco Live breakout sessions that will showcase this major new release on Thursday:

Cisco UCS Director 5.0
PSODCT-1004
Thursday, May 22
8:30 to 9:30am

Management and Automation of Application Centric Infrastructure (ACI) with Cisco UCS Director
BRKACI-2410
Thursday, May 22
12:30 to 2:00pm

In addition, we’ll be featuring live demos of UCS Director 5.0 all week in the Cisco Live World of Solutions expo. Here are some of the Cisco UCS Director + ACI demos you don’t want to miss:

Cisco Data Center and Cloud booth 

·    Infrastructure Automation and ACI with Cisco UCS Director

·    Cisco Prime Service Catalog + UCS Director + APIC

We also have several technology partners including NetApp, Nimble Storage, Sanovi, VCE, Zenoss and others that will be featuring integrated solutions with Cisco UCS Director.

But that’s not all. We’ve had several other recent new software releases including Cisco Intelligent Automation for Cloud version 4.0 for cloud management, Cisco Prime Service Catalog version 10.0 for enabling IT-as-a-Service, and Cisco Prime Network Services Controller version 3.2 for hybrid cloud workload mobility and other use cases. See below for some of the demos that will be featuring these and other Cisco data center management software products:

·   Dynamic Fabric Automation – including Cisco Prime Data Center Network Manager (DCNM) and Cisco Prime Network Services Controller

·   Virtual Network Services with Nexus 1000V – including Cisco Prime Network Services Controller

The sessions will cover Cisco’s Intercloud strategy and cloud solutions, Cisco IT’s internal cloud deployment, and other cloud computing topics. View the full agenda here.
